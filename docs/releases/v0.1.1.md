# OpenClaw 小助手 v0.1.1

这是首个对外可用的公开版本补丁发布，重点补了菜单栏可见性、实例自动发现、CLI 控制、忽略列表、管理页快捷入口，以及自动保活的稳定性问题。

## 亮点

- 菜单栏稳定显示运行状态
- 实例前置红绿状态圆点
- 一键打开每个实例的控制管理页
- 支持 CLI 批量控制实例
- 支持忽略实例和全量重新扫描
- 支持开机启动小助手
- 支持打开小助手时自动启动 OpenClaw
- 支持实例崩溃后自动重新拉起

## 本次修复

- 修复首屏菜单栏状态项可能不显示的问题
- 修复进程扫描时的死锁，避免状态刷新卡住
- 修复中文目录实例 ID 生成不稳定的问题
- 修复自动发现时丢失自定义 `--profile` / `--port` 参数的问题
- 修复实例曾手动停止后再次运行，自动保活不会重新接管的问题
- 改进外部启动实例的进程识别与端口推断

## 已验证

- 菜单栏 UI 可用
- CLI `status/start/stop/restart/remove/full-scan/set` 可用
- `launch-at-login` 开关可创建和移除 LaunchAgent
- `nexus-link` 在被强制杀掉后能被小助手自动拉起

## 已知事项

- 如果实例自己的 OpenClaw 配置无效，保活会持续重试，但实例仍会保持崩溃状态
- 这类情况需要先进入日志或实例仓库里修正配置
